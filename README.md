# FA Logistics

Send Love, Instantly, Anywhere. We Deliver.

1500+ curated gifts delivered to the US, Canada, UK, Europe and Asia. Every order is tracked, securely handled, and delivered on time.

## Setup

Serve the project directory with any static file server:

```
python3 -m http.server 8000
```

Open `http://localhost:8000` in a browser.

## Files

- `index.html` — Main application (single-page)
- `products.csv` — Product data
- `assets/` — Brand assets
